next技巧

发表于 2018-10-17 | 分类于 技巧

1. next主题的安装

最简单的安装方式就是直接克隆整个仓库

cd hexo
git clonegit clone https://github.com/theme-next/hexo-theme-next themes/next

2. next主题的更新

你可以通过如下命令更新到最新的master分支

cd themes/next
git pull

3. next主题的预览

💟 Muse 主题: LEAFERx | Alex LEE | Miaia
🔯 Mist 主题: Jeff | uchuhimo | xirong
♓️ Pisces 主题: Vi | Acris | Jiaxi He
♊️ Gemini 主题: Ivan.Nginx | Raincal | Dandy

4. 修改next主题

通过修改主题配置文件更改主题

Muse - 默认 Scheme，这是 NexT 最初的版本，黑白主调，大量留白
Mist - Muse 的紧凑版本，整洁有序的单栏外观
Pisces - 双栏 Scheme，小家碧玉似的清新
Gemini - 左侧网站信息及目录，块+片段结构布局

5. 创建主页

hexo new page ‘about’

在你的网站根目录下面的source文件夹会生成about文件夹

6. hexo博客添加搜索功能

标签和分类已经不能提供太大的作用，无法准确的定位到自己想要看的博客上去，所以添加一个本站内搜索功能是很有必要的

  • 安装插件

    npm install hexo-generator-searchdb –save

  • 修改站点配置文件

    search:
    path: search.xml
    field: post
    format: html
    limit: 10000

  • 修改主题配置文件

    local_search:
    enable: true

    7. 动态背景

    Canvas-nest
    three_waves
    canvas_lines
    canvas_sphere
    设置方法也很简单，直接设置里需要的动态背景为true。

markdown语法

发表于 2018-10-17 | 分类于 技巧

vscode 支持一边编辑一边预览，有2种方法：

  1. CTRL + shift + P 调出命令框，输入>markdown 找到markdown：open 》preview的选项选中
  2. 先按ctrl + k ,然后放掉，接着按 V ，调出实时预览框

markdown 语法

1. 标题

行首加上#号+空格 即可定义标题，#号的个数代表标题的级数，一共6级。

2. 列表

  • 无序列表

    • 可用星号（*），加号（+），减号（—）作为标记，后面跟空格。3者可混用。

  • 有序列表

    • 数字加上句号（1. ），数字不影响显示效果

      1. 11
      2. 222
      3. 22222

  • 列表的缩进和混排

    • 若需缩进，在*，+，—前面加 空格

  • 记事列表

    • 使用 - [x] 开头的表示已完成的记事
    • 使用 - [ ] 开头的表示待办记事
      • 完成记事
      • 待办记事

3. 嵌套块的引用

> 使用“>”可以表示嵌套的引用

4. 文字效果和分割线

斜体
粗体
++下划线++
删除线
==文本高亮==

一行中用三个以上的星号 (*)、减号 (-) 或是下划线 (_) 来建立一个分隔线，需要独占一行，中间可以有空格，但行内不能包含其他字符

5. 链接

  • 文本连接
    格式：[文本] (链接地址 “可选说明文字，鼠标放上提示的文字”)

    baidu

  • 图片链接
    格式：！[文本] (链接地址 “可选说明文字，鼠标放上提示的文字”)

6. 文字高亮和代码块

  • 需要高亮的文本，使用 号包括，例如高亮`
  • 引用代码块使用两组
